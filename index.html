I specialise in solving Partial Differential Equations using numerical methods. I am particularly fond of using OpenFOAM, an open source package for discretizing PDEs using Finite Volume Method. At present we are working on developing following modules in OpenFOAM: Shape Optimisation codes, Immersed Boundary Methods. We are also extensively using the newly developed OpenFOAM's overset mesh support to study designs of Vertical Axis Wind Turbines (VAWT).
